# Distribucion binomial con k exitos de n repeticiones, con probabilidad p.
def kbinomial(k,n,p):
    B = comb(n,k,exact=True)*(p**k)*((1-p)**(n-k))
    return B

# Distribucion de Poisson con k exitos y donde mu es la esperanza.
def kpoisson(k,u):
    P = math.exp(-u)*(u**k)/math.factorial(k)
    return P

Item 3¶

Primero defino una nueva funcin que simule una fuente que emite en promedio I = 15 fotones por segundo. Para esto se sigui la recomendacin de dividir un intervalo de tiempo (Dt) en m intervalos ms pequeos (dt) con m >> 1, aproximar la probabilidad de que la fuente emita un fotn como I * dt. Simulando la cantidad de fotones que se emitieron en cada dt, y sumando sobre todos los dt se obtuvo la emisin que realizaba la fuente en un determinado Dt (en este caso, 1 segundo).

Se despreci la probabilidad de emitir ms de un fotn en un intervalo. Como la distribucin de cada cuanto se emite un fotn es exponencial, si dt tiende a 0, la posibilidad de emitir 2 o ms se hace despreciable. Para comprobarlo calculamos las probabilidad de emitir 0, 1, 2 y 3 fotones en un dado dt. Adems sabemos que en cada dt tenemos un experimento de Bernoulli dado que la probabilidad de emitir un fotn en cada dt es independiente de lo que ocurri en los tiempos anteriores.

Luego se uso dicha funcin para repetir este experimento de emisin de la fuente unas 1000 veces, y con estos valores se construyo un histograma. Nuevamente se simul de forma terica la distribucin correspondiente (en este caso, Poisson con mu = I * Dt). Se la compar con una distribucin de Poisson porque p2 (probabilidad de emitir un fotn en dt) es muy chica pero m (divisiones de 1 segundo) es muy grande.

def fuente(I,Dt,m):
    dt=Dt/m
    p2=I*dt
    datos2=[]
    dt2=0
    while dt2 < Dt+dt:
        exitos, fracasos = experimento(1,p2)
        datos2.append(exitos)
        dt2 = dt2 + dt
    S = sum(datos2)
    return S

Item 6¶

En el nmero de datos de un determinado bin de un histograma esperamos una distribucin poissoneana siempre y cuando podamos decir que se realizaron muchos experimentos (como fue en todos estos casos) con una probabilidad baja. El problema con utilizar ete criterio se puede encontrar en casos donde se realizan pocos experimentos, y tambin en los bins que poseen mayor probabilidad.

Otro criterio que podra utilizarse es el de tomar una distribucin binomial para los datos de un bin de histograma, y estimar la probabilidad de caer dentro del bin como su tamao, y luego la probabilidad de caer fuera del bin (por eso binomial).

Edit: ahora sabemos que lo ideal sera una binomial de ms de dos casos (Multinomial!)
